# Ganesh Kisan Gaikwad - Professional Portfolio

A modern, responsive portfolio website showcasing my professional journey, projects, skills, and achievements in Electrical Engineering.

## 🌟 Features

- **Hero Section** - Eye-catching introduction with animated visuals
- **About Me** - Personal journey and career objectives
- **Professional Experience** - Internship details and accomplishments
- **Featured Projects** - Showcase of major projects and innovations
- **Skills & Expertise** - Technical and soft skills categorization
- **Education** - Academic journey and qualifications
- **Awards & Achievements** - Patents, research, certifications, and workshops
- **Contact** - Easy way to get in touch
- **Responsive Design** - Works perfectly on all devices
- **Dark/Light Theme** - Toggle between themes
- **Smooth Animations** - Professional transitions and effects

## 🛠️ Tech Stack

- **Frontend Framework**: React.js
- **Styling**: CSS3 with custom animations
- **Icons**: React Icons
- **Smooth Scrolling**: React Scroll
- **Responsive**: Mobile-first approach

## 🚀 Getting Started

### Prerequisites
- Node.js (v14.0.0 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/GaneshGaikwad007/portfolio.git
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## 📦 Build

To create a production build:
```bash
npm run build
```

## 🎨 Customization

### Colors
Edit the CSS variables in `src/index.css`:
```css
:root {
  --primary-color: #1e3c72;
  --secondary-color: #2a5298;
  --accent-color: #00d4ff;
  --text-light: #f4f4f4;
  --text-dark: #1a1a1a;
  --bg-dark: #0f0f0f;
  --bg-light: #f8f9fa;
  --border-color: #333;
}
```

### Content
Update the content directly in the component files or create a data file for easier management.
